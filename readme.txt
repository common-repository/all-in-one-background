All in one Background : You can specify a background image and color for a post / page.

== Description ==

* All in one Background : 포스트 / 페이지 배경이미지 및 색상을 지정할 수 있습니다. 
* 랜덤 슬라이드 메인 배경이미지, 특정일자의 배경이미지, 배경 패턴을 추가할 예정입니다.

== Installation ==

1. Upload the 'all-in-one-background' folder to the '/wp-content/plugins/' directory. 
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. 'All in one Background' post / page meta box added
